Corrosionwm is a wayland compositor written in rust that has the goal of being fast, customizable, safe, and beautiful. We endeavour to make our config as friendly as possible without sacrificing customizability. This window manager is in very very very very very early alpha, but doesn't tile yet, nor can it launch from tty TwT :( Any contributions to the corrosionwm source code would be greatly appreciated
