CorrosionWM is a wayland compositor written in rust that has the goal of being fast, customizable, safe, and beautiful. We endeavour to make our config as friendly as possible without sacrificing customizability. This window manager is currently in a state of very early alpha. It doesn't tile yet, nor can it launch from tty. Any contributions to the CorrosionWM source code would be greatly appreciated.

While you're here you should check out Poketube.
